샌드백 치기 로직
훈련 시작 안내 (2초)
    ↓
라운드 반복 (총 5라운드)
    ├── 랜덤 대기 (무입력 구간)
    ├── 방향키 신호 표시
    ├── 플레이어 입력 대기
    └── 판정 및 점수 계산
    ↓
결과 화면 → 공격력 경험치 지급

신호 생성 규칙
방향키 4종 (↑ ↓ ← →) 중 랜덤 10~15개 화면 오른쪽에서 왼쪽으로 오게 생성
직전 신호와 같은 방향 연속 출력 금지 (예측 방지)
신호 표시 지속 시간 : 500ms 고정
신호 간 대기 시간 : 800ms ~ 2500ms 랜덤
대기 중 입력은 전부 무효 처리 (미리 누르기 차단)
화면 중 특정 구간에서 플레이어 입력 확인
 
ex)
조건            |                              판정              |                        점수
올바른 키, 200ms 이내                  PERFECT                                            3점
올바른 키, 400ms 이내                   GOOD                                               2점
올바른 키, 500ms 이내                     OK                                              1점
500ms 초과 or 무입력                     MISS                                             0점
틀린 키 입력                                BAD                                          -1점

┌─────────────────────────────────┐
│      [ 샌드백 치기 ]        라운드 : 3/10        콤보 : 5🔥     │  ← 상단 정보바
├─────────────────────────────────┤
│                                                                  │
│                           [캐릭터]  VS  [샌드백]                 │  ← 메인 영역
│                                                                  │
├─────────────────────────────────┤
│                                     ←                           │  ← 신호 표시 영역
├─────────────────────────────────┤
│    점수 : 24                         PERFECT!                    │  ← 판정 텍스트
└─────────────────────────────────┘

달리기 로직
훈련 시작 안내 (조작법 표시, 2초)
    ↓
카운트다운 3 → 2 → 1 → GO!
    ↓
제한시간 동안 연타 (10~15초, 난이도에 따라 다름)
    ├── 키 입력 감지 → 유효 입력 카운트
    ├── 캐릭터 달리기 모션 업데이트
    └── 남은 시간 표시 업데이트
    ↓
시간 종료 → 결과 화면 → 체력 경험치 지급

입력 방식 (두 가지 중 하나)
A안 — 교대 연타 (A → D → A → D)
B안 — 스페이스 단일 연타

달리기 모션 로직
캐릭터 ASCII 프레임 2장을 교대로 출력
유효 입력이 들어올 때마다 프레임 전환
입력이 없으면 마지막 프레임 정지 유지
프레임 전환 속도가 곧 달리는 속도 시각화

예상 화면
┌──────────────────────────────────┐
│  [ 달리기 ]          남은 시간 : XX초       카운트 : XX            │  ← 상단 정보바
├──────────────────────────────────┤
│  
│                           캐릭터                     
│     
│                                               
├──────────────────────────────────┤
│     카운트 :                     A ↔ D  교대 입력(입력 가이드 메시지)                      
└──────────────────────────────────┘

반응속도 로직
훈련 시작 안내 (2초)
    ↓
5회 측정 반복
    ├── 랜덤 대기 (1000ms ~ 3500ms)
    ├── ★ or ✕ 출현
    ├── 플레이어 입력 감지
    └── 반응시간 기록 or 감점 처리
    ↓
5회 완료 → 유효 측정값 평균 계산
    ↓
결과 화면 → 공격속도 경험치 지급

신호 출현 규칙
화면의 지정 영역 내 랜덤 좌표에 출현
★ 출현 확률 75%, ✕ 출현 확률 25%
신호는 1500ms 동안 유지, 이후 자동 소멸
신호 소멸 후 미입력이면 해당 회차 MISS 처리
연속 ✕ 출현 최대 2회 제한 (3연속 함정 방지)

입력 판정 분기
★ 출현 시
스페이스 or 엔터 입력 → 반응시간 측정, 유효 기록
1500ms 내 무입력 → MISS, 해당 회차 측정값 제외

✕ 출현 시
무입력 유지 → 성공, 회차 보너스 +추가 점수
스페이스 or 엔터 입력 → 오반응, 해당 회차 측정값 무효 + 패널티 50ms 가산

반응시간 계산 방식
★ 출현 순간 타이머 시작
키 입력 순간 타이머 정지
측정값 저장

5회 중 MISS나 오반응으로 무효 처리된 회차 제외하고 평균 계산.
유효 측정값이 2회 미만이면 훈련 실패 처리 (경험치 0).
ex)
평균 반응속도                      공격속도 경험치
150ms 미만                                  15
250ms 미만                                  12
350ms 미만                                   8
500ms 미만                                   4
500ms 이상                                   1
